# 鼾静健康诊所 - 小程序真实全页面 UI 设计稿演示平台

本项目是一个独立的 **UI 设计稿演示项目**，由小程序端的 **53 个真实开发页面** 提取、打包并重新渲染而成。为了保证页面的独立性和防止样式冲突，本项目采用了 **多页面 iframe 架构**，将每一个小程序页面存储为独立的 HTML 文件。

---

## 🌟 核心特色功能

1. **📱 单机演示 (Device Simulator)**
   - 使用高度逼真、可无级缩放的 **iPhone 15 Pro Max 实体边框**，提供动态时钟与电量状态栏。
   - 内置 **WeChat 小程序原生导航栏 (Navbar)** 与 **底部栏 (Tabbar)** 模拟。
   - **交互式跳转**：支持微信底栏（首页、预约、治疗、商城、我的）的点击无缝切换，同时支持页面内部关键业务按钮的点击跳转逻辑，模拟真实小程序的用户路径。
   
2. **👥 双机比对 (Comparison View)**
   - 允许在左右两侧分别渲染两个独立手机，用来比对核心流程的上下文页面（如“睡眠量表”与“评估报告”、“就诊预约”与“预约成功”）。

3. **🗺️ 平铺图纸 (Figma Grid View)**
   - Figma 风格的全局平铺，以 53 个微型画板的形式展示所有设计稿，适合全局把控，点击任意画板可直接载入模拟器中查看详情。

4. **🛠️ 代码标注模式 (Inspect Spec Mode)**
   - 切换右上角“**代码标注模式**”后，可在手机屏幕中悬浮选中任意 UI 元素。
   - 点击元素可在右侧标注面板中，实时读取其 **HTML 标签名**、**CSS 类名**、**占位尺寸** 以及 **Computed CSS 样式规则**。

5. **🔍 模糊搜索 & 缩放 (Search & Scaling)**
   - 支持根据页面标题（如“分销”）或路由路径（如`pages/order`）进行实时的侧边栏搜索过滤。
   - 内置无极缩放拉条，可在 `40% ~ 130%` 之间任意调整设备大小，适应不同屏幕尺寸。

6. **🌗 主题背景切换 (Dark/Light Shell)**
   - 可在磨砂暗黑玻璃拟态（Default）与高雅亮白主题之间一键切换背景，适配不同展示环境。

## 🚀 启动与运行方式

本项目采用无网络依赖设计，提供以下多种启动方式：

### 方案 A：双击即用（零配置推荐）
1. 进入 `ui` 目录。
2. 直接双击 `index.html` 用浏览器打开即可。
> 💡 *注意：由于所有数据均已本地化，直接以 `file://` 协议运行不会遇到任何跨域问题，适合离线查阅页面。*

### 方案 B：启动极简本地静态服务（推荐体验完整功能）
如果需要解锁右上角的“**代码标注模式**”（因浏览器安全策略，直接双击打开的 `file://` 协议会禁止读取 iframe 内部元素）：
```bash
npm run serve
# 或者直接运行: npx -y http-server . -p 8080 -o
```
此服务将自动在浏览器中打开 `http://localhost:8080`，此时可自由选中模拟器中的任意文本、图片或块卡片查看完整的 CSS 样式标注。

### 方案 C：使用前端开发服务器 (Vite)
如果你想集成到前端脚手架或利用 Vite 挂载：
```bash
# 1. 安装开发依赖
npm install

# 2. 启动开发服务器
npm run dev
```

## 🎨 设计规范

本项目与小程序端共用一套 Design Tokens，可在 `css/app-styles.css` 中查找以下变量：
- **品牌主色 (舒眠蓝)**: `--color-primary-50` ~ `--color-primary-800` (主色为 `#3B6BF5`)
- **辅助健康色 (安宁绿)**: `--color-secondary-100` ~ `--color-secondary-900`
- **分销专属 (暖金)**: `--color-gold-400`
- **中性灰色度**: `--color-gray-50` ~ `--color-gray-900`
- **标准卡片圆角**: `--radius-md` (12px), `--radius-lg` (16px)
- **标准投影**: `--shadow-md`
