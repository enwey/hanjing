# 鼾静健康诊所 - 使用与培训手册

本手册分为三大部分：管理员/医生后台操作手册、小程序端用户操作教程、以及临床客服与咨询师的培训培训材料。

---

## 1. 管理员操作手册 (Admin Operations Manual)

管理后台是诊所运营和前台接诊的核心入口。

### 1.1 医生排班管理 (Doctor Scheduling)
1. 登录管理员后台，进入 **[排班管理]** 菜单；
2. 选择要排班的 **[门店]**、**[科室]** 与 **[医生姓名]**；
3. 点击日历上的具体日期格子：
   * 可设置该日为 **[出诊]** 或 **[休息]**；
   * 出诊需选择绑定的物理门店，并设定 **[上午出诊容量]** 和 **[下午出诊容量]**。
4. **批量排班**：支持一键复制“上月排班数据”快速填充本月排班，提高前台排班效率。
5. **超卖拦截保障**：系统在底层对已预订号源做锁行防护，若某时段已被患者约满，后台将拦截“改为休息”的指令，防止医疗事故。

### 1.2 就诊记录录入与病历归档
1. 护士接诊登记：在 **[今日接诊]** 看板选中已到店的患者，点击 **[到店分诊]**，录入身高、体重、血压、颈围等临床前检（Pre-exam）数据。系统会自动计算 **BMI** 供医生诊断时参考；
2. 医生接诊：点击 **[开始就诊]**，在就诊表单中录入：
   * 主诉及主要症状描述；
   * 诊断结论；
   * 治疗方案建议（如阻鼾器适配调咬合等）。
3. 上传体检单或口腔全景片附件，点击 **[确认就诊完毕]** 自动归档至患者电子病历，数据永久留痕。

### 1.3 预约改约与退款处理规范 (Rescheduling & Refund Policy)
当改约后的挂号费或预约定金总额与原预约金额不一致时，为确保系统账单与微信支付财务对账完全一致，必须遵循以下业务规范：
1. **推荐执行“取消并重新预约”流程**：
   - 引导患者在小程序端取消原预约，或由管理员在后台操作取消；
   - 系统将自动对该预约关联的定金订单发起退款流程（状态变更为 `refunding`），并在此笔**定金支付成功满 3 日后**，由后台自动执行原路退款（状态变更为 `refunded`）；
   - 随后由患者重新在小程序端预约并支付新金额。
2. **禁止直接后台改约不同金额的医生**：
   - 若管理员直接在后台将预约调整为不同挂号费的医生，系统虽会修改预约记录上的诊查费数值，但并不会自动发起任何线上资金的多退少补。若已发生此操作，必须通过线下门诊前台收银台手动补缴差额，或在微信商户后台手动操作差额退款。

## 2. 用户操作教程 (User Tutorial)

用户通过微信小程序实现首诊预约、筛查评测以及智能设备佩戴自控。

### 2.1 微信小程序首诊挂号流程
1. 打开“鼾静健康诊所”小程序首页，点击 **[门诊预约]**；
2. 根据地理位置选择最近的 **[诊所门店]**，查看该门店下各位医生的资质详情与执业特长；
3. 选择就诊日期与就诊人段（上午/下午），选择 **[就诊人信息]**（支持绑定家人档案）；
4. 勾选未成年人拦截提示（未成年人挂号阻鼾器类目会拉起专科科普并强警告拦截）；
5. 支付预约定金，支付成功后系统自动锁定排班额度，并推送挂号挂号通知。

### 2.2 会员积分与兑换操作
1. 完成实名认证（输入真实姓名及18位身份证号）后，会员中心自动赠送 **“实名认证专享体验积分包”（2000 积分）**；
2. 在 **[积分商城]** 选中所需要代金券（如“黄金会员专享100元券”、“洁泡腾片10元券”）；
3. 点击 **[积分兑换]**，系统将自动扣减账户积分，代金券自动发放至 **[我的卡包]** 供商城下单或线下核销。

## 3. 客服与咨询师培训材料 (Training Materials)

客服及前台咨询师在接待打鼾患者时，必须掌握以下专业医学逻辑。

### 3.1 阻塞性睡眠呼吸暂停 (OSAHS) 与阻鼾器逻辑
* **病理特征**：患者在深睡时咽部肌肉松弛，舌根后坠，气道受到部分或全部压迫，引发强烈打鼾并伴有窒息（呼吸暂停）。
* **阻鼾器原理**：下颌前移阻鼾器 (MAD) 属于物理微调矫治器。患者夜间佩戴时，器件通过力学原理将患者的下颌向前牵引 5mm~10mm，带动舌根部向前位移，拉紧咽后壁肌肉，从而在物理上强行维持咽部气道呈开放状态，消除打鼾及窒息。

### 3.2 未成年人就诊拦截的医学考查 (Adenoid Hypertrophy)
* **核心禁忌**：**严禁为未满18周岁的儿童/青少年直接适配下颌前移阻鼾器**。
* **宣教口径**：儿童处于颌骨及牙列生长发育的关键期，强行佩戴前移矫治器会导致下颌骨过度发育（地包天）及咬合畸形。
* **诊断建议**：儿童打鼾的大部分病因在于 **腺样体肥大（Adenoid Hypertrophy）** 或 **扁桃体肥大**，导致上呼吸道物理阻塞。咨询客服应强烈建议家长带孩子前往三甲医院耳鼻喉科进行喉镜或侧位 X 线片排查，不可盲目佩戴成人阻鼾器。
